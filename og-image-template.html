  1. Open this HTML file in your browser (Chrome recommended)
  2. Press F12 to open Developer Tools
  3. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
  4. Type "screenshot" and select "Capture node screenshot"
  5. Click on the blue gradient box (the OG image)
  6. The image will download automatically as og-image.png
  7. Move the downloaded image to your public/ folder
  8. Rename it to og-image.png if needed
